# expense-tracker
Simple expense tracker application to manage your finances.

Features:
Users can add an expense with a description and amount.
Users can update an expense.
Users can delete an expense.
Users can view all expenses.
Users can view a summary of expenses for a specific month (of current year).
Users can view a summary of expenses for a specific category.

Instalation
```git clone https://github.com/SalikNikodem/expense-tracker.git```

Usage
add --description --amount --category
delete --id
view --month(optional) --category(optional)
update --id (req) --descritpion --amount --date --category (optional, at least one required)
